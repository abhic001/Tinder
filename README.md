# Tinder

It is a clone of Tinder app built to understand the basics of Tinder's card UI, animation and rotation.

## Features Implemented

- [x] Draggable Card
- [x] Rotation on Drag
- [x] Final Card Animation
- [x] Basic Modal Transition
